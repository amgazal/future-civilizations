This repository contains the official website for the **Future Civilizations Society** at Cornell University.  
The site serves as a public-facing hub for the society and its flagship conference, which explores how artificial intelligence, autonomous systems, and emerging technologies will reshape human civilization and future cities.

## About the Project
The Future Civilizations Conference is a multidisciplinary initiative bringing together students, faculty, researchers, and industry leaders to examine questions around autonomous cities, sustainability, social psychology, and frontier settlements such as Mars.

The website is designed to be lightweight, fast, and easy to maintain as the conference evolves.

## Features
- Responsive, mobile-first design using pure HTML, CSS, and JavaScript
- Thematic panels outlining key conference focus areas
- Dedicated conference page with countdown and event overview
- Executive board section with profile images
- Hosted on GitHub Pages for easy deployment and updates

## Tech Stack
- HTML5
- CSS3 (custom styling, no frameworks)
- Vanilla JavaScript
- GitHub Pages (hosting)

##  Deployment
The site is deployed using **GitHub Pages** and automatically updates on commits to the `main` branch.
